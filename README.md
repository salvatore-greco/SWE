# Progetto di Ingegneria del software anno accademico 2024/25
Idea di progetto:\
Sistema di gestione di una biblioteca comunale -> Prestito dei libri, visione dei libri disponibili, prenotazione del prestito, gestione sale studio con gestione presenze posti disponibili, prenotazione posti, gestione eventi (tipo le presentazioni dei libri). Sportello al cittadino

Attori:
- Bibliotecari 
- Cittadini (quelli che leggono i libri e che vogliono sapere che eventi ci sono in biblioteca)
- Studenti (utilizzatori di aule studio)

Collegare il database per autenticazione, gestione libri
